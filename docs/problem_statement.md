# ❗ Problem Statement

Temperature monitoring is a fundamental requirement in various domains such as environmental observation, smart homes, agriculture, and industrial systems. Traditional temperature measurement methods often require manual observation and lack real-time accessibility, making them inefficient for continuous monitoring.

In many cases, there is a need for a system that can:
- Measure temperature automatically
- Provide continuous and real-time updates
- Allow remote access to the data
- Be low-cost and easy to implement

Conventional systems either lack wireless communication capabilities or require expensive infrastructure, making them less suitable for small-scale or personal applications.

Therefore, the challenge is to design a compact, cost-effective, and efficient system that can monitor temperature in real-time and make the data accessible remotely through an intuitive interface.
