# 💡 Motivation

The rapid growth of IoT (Internet of Things) has enabled the development of smart systems capable of monitoring and controlling environments remotely. Temperature monitoring, being one of the most common sensing applications, provides an excellent opportunity to explore the integration of embedded systems with cloud-based platforms.

This project was motivated by the need to understand how low-cost hardware components like the ESP8266 and DHT11 sensor can be combined to build a real-time monitoring system. It also aims to demonstrate how wireless communication and cloud dashboards can enhance accessibility and usability.

Additionally, the project serves as a practical implementation of concepts learned in embedded systems and IoT, including sensor interfacing, microcontroller programming, and data transmission over the internet.

By building this system, the goal was to:
- Gain hands-on experience with IoT technologies  
- Understand real-time data acquisition and transmission  
- Explore integration with cloud-based visualization tools (Blynk)  
- Develop a scalable foundation for future smart monitoring systems  

Overall, this project reflects an effort to bridge theoretical knowledge with practical application in the field of IoT and embedded systems.
