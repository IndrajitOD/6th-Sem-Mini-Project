# 📘 Project Overview

- This project presents an IoT-based temperature monitoring system using the ESP8266 NodeMCU and DHT11 sensor. The system is designed to measure ambient temperature in real-time and display the data remotely through a mobile dashboard using the Blynk IoT platform.

- The DHT11 sensor acts as the data acquisition unit, capturing temperature values from the surrounding environment. These values are read and processed by the ESP8266 microcontroller, which also handles wireless communication through its built-in Wi-Fi capability.

- The processed data is transmitted to the Blynk cloud server, where it is visualized on a user-friendly dashboard. The dashboard displays temperature readings in both Celsius and Fahrenheit using virtual pins, allowing users to monitor the data in real-time from a smartphone.

- The system operates continuously by updating the sensor readings at regular intervals, ensuring near real-time monitoring. This project demonstrates the integration of embedded systems with IoT platforms to create a simple yet effective remote monitoring solution.

- Overall, the project highlights how low-cost hardware combined with cloud-based applications can be used to build scalable and efficient IoT systems.
